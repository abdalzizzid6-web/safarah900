# تقرير مصادر بيانات API (Football Data Sources)

يقوم النظام حالياً بدعم ثلاث مصادر بيانات رئيسية لتحسين مرونة التغطية الرياضية:

1. **API-Football (المصدر الافتراضي):**
   - **الاستخدام:** المصدر الأساسي للمباريات، النتائج، المجموعات، والفرق.
   - **المميزات:** توفير تغطية واسعة، تحديثات لحظية (Live Scores)، وبيانات تفصيلية للمباريات.
   - **حالة الربط:** مستقر وجاهز.

2. **SportMonks:**
   - **الاستخدام:** مستخدم كبديل عالي الأداء مع تحكم أدق في البيانات.
   - **المميزات:** استجابة سريعة، بيانات إحصائية دقيقة، وهيكلية بيانات سهلة المعالجة.
   - **حالة الربط:** يتطلب إعداد مفتاح API خاص.

3. **TheSportsDB:**
   - **الاستخدام:** مصدر إضافي لبيانات الفرق والشعارات ومعلومات الدوريات الأساسية.
   - **المميزات:** مجاني للمعلومات الأساسية، قاعدة بيانات مجتمعية غنية.

---
### التحديثات على لوحة التحكم:
لرفع مستوى الاحترافية، تم تحديث لوحة التحكم (`WcAdmin`) لإضافة قسم **"إعدادات مزودي البيانات"** يتيح:
- تحديد المصدر المفضل لكل مجال (مباريات، دوريات، فرق، الخ).
- إدارة مفاتيح الـ API (API Keys) مباشرة من اللوحة.
- اختبار الاتصال بكل مصدر قبل الحفظ للتأكد من صحة المفاتيح.
- حفظ الإعدادات في Firestore لتطبيقها على النظام بالكامل.
